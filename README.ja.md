# vscode-all-in-one-icon

<!-- Plugin description -->

JetBrains IDE に美しい VS Code スタイルのファイル・フォルダアイコンを追加するプラグインです。VSCode Icons、Material Icons、File Icons など複数のテーマに対応しています。

**特徴:**

- 複数のアイコンテーマ（VSCode Icons, Material Icons, File Icons）
- テーマ優先順位システム（フォールバック対応）
- 2,000以上のファイル・フォルダアイコン
- 多言語UI（English, 日本語, 简体中文, 한국어）
- 簡単なテーマ切り替え
<!-- Plugin description end -->

![Build](https://github.com/yuyu1815/vscode-all-in-one-icon/workflows/Build/badge.svg)

## 機能

- **複数のアイコンテーマ**
  - **VSCode Icons**: 人気の [vscode-icons/vscode-icons](https://github.com/vscode-icons/vscode-icons) プロジェクトベース
  - **Material Icons**: マテリアルデザインのアイコン
  - **File Icons**: [file-icons/vscode](https://github.com/file-icons/vscode) 由来の2,000以上のアイコン

- **幅広いカバレッジ**
  - 100以上のファイル拡張子に対応
  - 人気のフレームワーク・ツール用の特別アイコン
  - 一般的なディレクトリ用のカスタムフォルダアイコン

- **簡単設定**
  - シンプルなプロジェクトレベル設定
  - プロジェクト毎にカスタムアイコンの有効/無効切り替え
  - 再起動なしでテーマを即時切り替え

## インストール

### JetBrains Marketplace から

1. JetBrains IDE を開く
2. <kbd>ファイル</kbd> > <kbd>設定</kbd> > <kbd>プラグイン</kbd>
3. <kbd>Marketplace</kbd> をクリック
4. 「vscode-all-in-one-icon」を検索
5. <kbd>インストール</kbd> をクリック

### 手動インストール

1. [GitHub Releases](https://github.com/yuyu1815/vscode-all-in-one-icon/releases/latest) から最新版をダウンロード
2. <kbd>ファイル</kbd> > <kbd>設定</kbd> > <kbd>プラグイン</kbd>
3. 歯車アイコン ⚙️ をクリック
4. <kbd>ディスクからプラグインをインストール...</kbd> を選択
5. ダウンロードしたファイルを選択

## 設定

プラグインインストール後:

1. <kbd>ファイル</kbd> > <kbd>設定</kbd> > <kbd>ツール</kbd> > <kbd>Icon Theme Settings</kbd>
2. **言語**: UIの言語を選択
3. **テーマ優先順位**:
   - 「有効なテーマ」に優先順位順でテーマが表示
   - ▲/▼ ボタンで順序変更
   - 追加/削除 ボタンでテーマを管理
   - アイコンは上のテーマから検索され、見つからなければ次のテーマにフォールバック
4. 「カスタムファイルアイコンを有効にする」でプラグインの有効/無効を切り替え
5. <kbd>適用</kbd> で設定を保存
